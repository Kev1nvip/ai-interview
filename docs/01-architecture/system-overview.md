# 系统总览

## 1. 项目目标

构建一个本地运行的 AI 面试系统，用户可通过语音与 AI 面试官互动，AI 面试官通过 LLM 提问、追问、总结与评分，并通过 TTS 输出语音，前端以 Streamlit + Live2D 方式展示。

---

## 2. 系统模块

系统分为三大模块：

### 2.1 LLM模块
负责：
- 面试官人设控制
- 问题生成
- 多轮追问
- 面试总结
- 候选人表现评分

### 2.2 语音交互模块
负责：
- 麦克风输入
- 语音识别（ASR）
- 语音合成（TTS）
- 音频缓冲与播放控制

### 2.3 页面集成模块
负责：
- Streamlit 页面交互
- 展示对话历史
- 触发录音与播放
- Live2D 角色展示与口型联动

---

## 3. 核心数据流

用户说话
-> 前端采集音频
-> ASR转文字
-> LLM生成面试官回复
-> TTS将回复转语音
-> 前端播放音频
-> Live2D同步动作/口型

---

## 4. 模块边界

### LLM组输入
- 用户文本
- 当前轮次上下文
- 面试阶段

### LLM组输出
- 面试官文本回复
- 当前阶段标签
- 是否继续追问
- 是否结束面试

### 语音组输入
- 用户音频
- 面试官回复文本

### 语音组输出
- 识别文本
- 合成音频文件或音频流

### 前端组输入
- 用户音频流
- ASR文本
- LLM文本
- TTS音频

### 前端组输出
- 页面交互状态
- Live2D动画状态
