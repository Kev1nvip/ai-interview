# LLM模块说明

## 目标
负责构建 AI 面试官的大脑能力，包括提问、追问、总结与评分。

## 负责人
- 成员1：模型接入与推理性能
- 成员2：提示词设计与面试逻辑

## 工作内容
1. 接入 Qwen3.5-4B
2. 设计系统提示词
3. 设计多轮对话管理逻辑
4. 输出结构化面试结果
5. 支持总结与评分

## 输入
- 用户文本
- 当前阶段
- 对话历史

## 输出
- 面试官回复文本
- 状态控制信息
- 评分结果
