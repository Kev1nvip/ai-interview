# Qwen3.5-4B 集成说明

## 1. 模型
- 模型名称：Qwen3.5-4B
- 运行方式：本地推理
- 建议加载方式：llama-cpp-python / transformers（二选一，优先前者）

## 2. 建议配置
- temperature: 0.7
- top_p: 0.8
- max_tokens: 256
- context_length: 4096

## 3. 最小验证目标
输入：
“请你作为AI面试官，要求候选人进行自我介绍”

输出：
“你好，欢迎参加本次面试。请先做一个简短的自我介绍。”

## 4. 风险点
- 输出不稳定，可能偏离结构化格式
- 长对话时上下文过长
- 面试阶段切换可能不准确

## 5. 解决方向
- 使用固定输出模板
- 对回复做后处理
- 限制每轮历史长度
