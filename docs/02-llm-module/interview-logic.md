# 面试逻辑设计

## 1. 问题类型

### 开场问题
- 欢迎语
- 面试流程介绍

### 自我介绍问题
- 请做一个简短自我介绍

### 技术问题
- 与岗位相关的核心基础知识
- 框架与工程能力问题

### 项目追问
- 项目背景
- 职责分工
- 技术难点
- 性能优化
- 故障处理

### 行为问题
- 团队协作
- 冲突处理
- 压力场景

### 总结问题
- 候选人是否有反问

---

## 2. 追问逻辑

追问条件：
- 候选人提到具体项目
- 候选人回答过于泛化
- 候选人回答中存在明显逻辑缺口

不追问条件：
- 当前问题已回答充分
- 问题轮次已超过限制
- 用户明确表示不清楚/不了解

---

## 3. 结束逻辑

以下情况可触发结束：
- 问答轮次达到上限
- 用户主动结束
- 已覆盖主要面试阶段
